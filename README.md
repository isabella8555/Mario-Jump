🍄 Mario Jump

Um jogo simples inspirado no Mario, feito com HTML, CSS e JavaScript.

Sobre o jogo

No jogo, o personagem precisa pular os obstáculos que aparecem na tela.
Se encostar no obstáculo, o jogo termina.

Funcionalidades

Sistema de pulo

Detecção de colisão

Animações com CSS

Reinício ao perder

Tecnologias usadas

HTML

CSS

JavaScript

Objetivo

Projeto desenvolvido para praticar lógica de programação, manipulação do DOM e animações com JavaScript.

Como jogar

Clone ou baixe o repositório

Abra o arquivo index.html no navegador

Use a tecla de espaço (ou clique) para pular
